# 📚 Kutuphanee

1\. sınıfta geliştirdiğim kişisel kütüphane takip uygulaması. Kitaplarını ekleyip yönetebilir, notlar alabilir ve favorilere ekleyebilirsiniz.

---

## ✨ Özellikler

- Kitap ekleme (başlık, yazar, tür)
- Yazara göre filtreleme
- Kitap detay sayfasında not ekleme
- Favorilere ekleme / çıkarma
- Tüm notları görüntüleme ve düzenleme

---

## 🛠 Kullanılan Teknolojiler

- Java
- Android Studio
- SQLite (yerel veritabanı)
- ListView + BaseAdapter (ViewHolder pattern)

---

## 📝 Öğrendiklerim

Bu proje Android geliştirmeye giriştiğim ilk ciddi denemem. SQLite ile CRUD işlemleri yapmayı, Activity'ler arası veri geçişini (`Intent`) ve adapter pattern'i bu projede öğrendim. Kodun bazı bölümlerini bugün farklı yazardım ama öğrenme sürecinin bir parçası olarak burada duruyor.
